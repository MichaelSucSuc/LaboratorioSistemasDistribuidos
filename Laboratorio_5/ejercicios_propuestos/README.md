Ejercicios propuestos

1) Servicio RPC Tradicional (Java RMI)

Archivos de ejemplo (lista para copiar/pegar):
- rmi/Calculator.java
- rmi/CalculatorImpl.java
- rmi/Server.java
- rmi/Client.java

Compilar/ejecutar: compilar con `javac` y ejecutar el `Server` y `Client` en terminales separadas.

2) Sistema de Conversión con gRPC (Java)

Archivos de ejemplo:
- converter/converter.proto
- converter/ConverterServer.java

Requisitos y tareas del estudiante:
- Añadir más conversiones (ej. F->C, Dólares->Soles, Millas->Km).
- Validar entradas (valores negativos donde no correspondan).
- Mostrar logs del servidor.

Comparativa (entregable): ver `comparativa.md`.

Entrega:
- Código fuente cliente/servidor
- Archivo .proto
- Evidencias de ejecución (capturas o logs)
- Tabla comparativa de resultados
- Respuestas argumentadas del cuestionario
